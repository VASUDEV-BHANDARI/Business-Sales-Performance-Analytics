# 📊 Business Sales Performance Analytics (Power BI)

## 🔍 Project Overview
This project analyzes business sales data to provide actionable insights for decision-making. Using Power BI, it visualizes sales trends, product performance, and regional profitability to help businesses identify opportunities and risks.

## 🛠️ Tools Used
- Power BI  
- Microsoft Excel  
- DAX (Data Analysis Expressions)  

## 📊 Key Features
- KPI Cards: Total Sales, Profit, Orders, Quantity  
- Sales Trend Analysis (Monthly/Yearly)  
- Category & Segment Performance  
- Region-wise Sales Analysis  
- Top Selling & Loss-Making Products  
- Discount vs Profit Analysis  

## 💡 Insights & Recommendations
- Identified high-profit categories and regions for strategic focus  
- Highlighted loss-making products affecting profitability  
- Analyzed the impact of discounts on profit  
- Recommendations: focus on profitable categories, reduce excessive discounts, improve low-performing regions  

## 🎯 Learning Outcomes
- Data Cleaning & Preparation  
- Data Visualization & Dashboard Design  
- Business Insight Generation  
- Real-world Analytics Problem Solving
